# icecream-backend

Back-End

## 실행 방법

### 개발 환경

- macOS 11.2
- Git 2.31
- Yarn 1.22
- Docker 20.10
- Docker Compose 1.28
- Visual Studio Code 1.54

### 프로덕션 빌드 및 실행

```shell
> yarn build
> docker-compose up --detach
```

위 명령어를 실행하기 전에 기존에 실행 중인 컨테이너를 중지하고, 생성했던 이미지도 삭제한다.
